HTML significa:

→ HyperText Markup Language (Linguagem de Marcação de Hypertexto).

    • Linguagem: Forma de comunicação. No nosso caso, entre o programador e o navegador de Internet (Google Chrome, Firefox, Safari, Edge, Internet Explorer…)
      Marcação: O HTML 'marca' o que é o que em uma página. Ele que vai dizer o que é um texto, o que é uma imagem, um vídeo e assim por diante.
      Hypertexto: Hypertexto faz referência a escritá gráfica, sintetizando, é um nome bonito para site.

    • Resumindo…HTML é a tecnologia que fala o que é cada item em uma página de internet / um site. E ele fez isso por meio de TAG's.

    TAG's ? Tag's são etiquetas. São os trechos de código que nos ajudam a falar o que é o que em uma página.

    Principais Tags HTML

    Esta página mostra algumas das tags mais usadas no HTML com explicações e exemplos práticos.

    <html>

    É a tag raiz que envolve toda a página HTML.

    <head>

    Contém informações sobre a página, como título e dados de configuração.

    <title>

    Define o título da página que aparece na aba do navegador.

    <body>

    Contém todo o conteúdo visível da página, como textos, imagens e links.

    <h1> a <h6>

    Tags de título, de <h1> (mais importante) até <h6> (menos importante).

    <p>

    Usada para criar parágrafos de texto.

    <br>

    Insere uma quebra de linha.
    Exemplo: este texto está na linha seguinte.

    <hr>

    Insere uma linha horizontal para separar seções.

    <strong> e <em>

    <strong> dá ênfase forte (negrito) e <em> dá ênfase (itálico).

    Exemplo: Texto em negrito e texto em itálico.

    <a>

    Cria links para outras páginas ou locais da web.

    Exemplo: Visitar o Google

    <img>

    Insere imagens na página. Usa o atributo src para definir o caminho da imagem.

    <ul> e <li>

    Cria listas não ordenadas.

    • Item 1
    • Item 2
    • Item 3

    <ol> e <li>

    Cria listas ordenadas (numeradas).

    1. Primeiro
    2. Segundo
    3. Terceiro

    <div> e <span>

    <div> cria uma seção de bloco, e <span> aplica estilo em linha.

    Soy uma <div>

    Texto com parte em vermelho usando <span>.

    <table>

    Usada para criar tabelas.

    Nome Idade
    Maria 25
    João 30

    <form>, <input>, <button>

    Usadas para criar formulários e capturar dados do usuário.



    <video> e <audio>

    Permitem inserir vídeos e áudios diretamente na página.


    <iframe>

    Incorpora outra página da web dentro da sua.

    <footer>

    Define o rodapé da página.
