JavaScript

Voltar

JavaScript é uma linguagem de programação usada para tornar páginas web interativas. Ela permite manipular elementos do HTML, responder a ações do usuário e criar aplicações dinâmicas no navegador.

alert()

Exibe uma mensagem em uma janela de alerta.

alert("Olá, mundo!");

console.log()

Mostra mensagens no console do navegador (usando inspecionar no navegador...).

console.log("Mensagem no console");

Variáveis

Declaramos variáveis com var, let ou const.

let nome = "Jean";
const idade = 25;

Tipos de Dados

Principais tipos: string, number, boolean, object.

const texto = "Olá";
const numero = 10;
const booleano = true;
